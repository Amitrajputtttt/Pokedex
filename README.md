# Pokedex

A web application to search for and view details about different Pok√©mon. Built using HTML, CSS, and JavaScript.

## Installation

To get a local copy up and running, follow these simple steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/Amitrajputttt/Pokedex.git
    ```
2. Open the project directory:
    ```bash
    cd Pokedex
    ```
3. Open `index.html` in your favorite browser to start the application.
